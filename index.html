Guide

Getting Started

  • Place Nodes: Click anywhere on the canvas to create nodes
  • Spawn Individuals: Select a node and click "Spawn Individual" when you have enough food
  • Watch Evolution: Individuals will automatically search for food and return to the colony

Controls

  • Right-click: Access node actions menu
  • Hover: View detailed information in tooltips
  • Speed Slider: Adjust simulation speed (0.1x to 5x)
